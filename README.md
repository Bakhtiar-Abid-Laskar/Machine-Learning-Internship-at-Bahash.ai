## üìö Curriculum & Projects

The internship progressed through the following core modules:

### 1. Introduction & Basics
* **Basics of ML:** Understanding the fundamental concepts of Machine Learning.
* **Data Preprocessing:**
    * Handling categorical data using **Dummy Variables & One Hot Encoding**.
    * Splitting datasets into **Training and Testing** sets.
* **Model Persistance:** Saving and loading trained models using **Joblib** and **Pickle**.

### 2. Supervised Learning: Regression
* **Linear Regression (Single Variable):** Predicting outcomes based on a single feature.
* **Linear Regression (Multiple Variables):** Working with multivariate data.
* **Gradient Descent:** Implementing the core optimization algorithm and understanding the Cost Function.
* **Regularization:** Reducing overfitting using **L1 (Lasso)** and **L2 (Ridge)** Regression.

### 3. Supervised Learning: Classification
* **Logistic Regression:** Binary classification problems.
* **Decision Trees:** Building tree-based models for decision making.
* **Support Vector Machine (SVM):** Implementing SVM for effective classification boundaries.
* **Random Forest:** Using ensemble methods to improve accuracy.
* **Naive Bayes Classifier:**
    * Part 1: Concepts and Probability.
    * Part 2: Implementation (Gaussian/Multinomial).
* **K-Nearest Neighbors (KNN):** Classification using instance-based learning.

### 4. Unsupervised Learning
* **K-Means Clustering:** Partitioning data into distinct groups based on similarity.
* **Principal Component Analysis (PCA):** Dimensionality reduction techniques for optimizing performance.

### 5. Model Evaluation & Optimization
* **K-Fold Cross Validation:** robustly estimating model performance.
* **Hyperparameter Tuning:** Optimizing models using **GridSearchCV**.
* **Bias vs. Variance:** Understanding the trade-off to minimize errors.
* **Ensemble Learning:** Improving model stability using **Bagging**.
